Puti Wen

Neuroimaging Scientist & Clinical Tool Builder · Abu Dhabi, UAE

A somewhat unconventional neuroscientist. My PhD was in visual neuroscience, but my recent work has centered on building diagnostic and prognostic reporting tools for multiple sclerosis. I also spend a lot of time building processing pipelines and maintaining infrastructure for large neuroimaging datasets. I work across mixed fields: psychology, radiology, engineering, clinical neurology, or whatever the next problem requires.

Open to roles at the intersection of neuroimaging, clinical software, and scientific infrastructure.

Projects

Things I've built

Clinical Tool

LesionView

Clinical imaging app for tracking MS lesions across timepoints. Automated longitudinal reports identify new and enlarging lesions with color-coded and animated visualization. Co-inventor on provisional patent (NYU Ref: ABD01-03).

React Native / Expo / MATLAB
AI / ML

CvsView

Web-based review tool for clinicians to verify the central vein sign in MS lesions. Includes a modular, plug-and-play AI component for future models. Docker-containerized and deployed as an on-demand clinical web service.

Docker / Machine Learning / Python
Infrastructure

XNAT & Brain Imaging Biobank

End-to-end management of the XNAT neuroimaging platform. Built six containerized processing pipelines on HPC and automated daily validation dashboards and backup workflows for the brain imaging biobank.

Docker / Singularity / HPC / GitHub Actions
Full-Stack App

MyPhysio

Full-stack clinical app for physiotherapy practice management: admin dashboards, patient portals, exercise prescription, and pain and range-of-motion tracking.

React Native / Next.js / Supabase
Visualization

afqView

Interactive 3D white matter tractography QC tool. Side-by-side before/after tract comparison, per-subject navigation, and color-coded availability indicators across 100+ subjects and 20+ tracts.

Three.js / Node.js / Vite
Data Science

Normative Brain Dashboard

Preprocessed 9,000+ subjects from HCP and ADNI datasets. Built a normative dashboard with age/gender-controlled brain structure metrics, allowing researchers to compare individual data against population curves.

Python / MATLAB / Data Visualization
