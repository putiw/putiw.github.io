Key projects

XNAT Platform & Brain Imaging Biobank

NYUAD Center for Brain and Health

  • Processing pipelines: Built and containerized six pipelines on HPC for functional MRI preprocessing, structural brain segmentation, diffusion tractography, MRI quality control, cortical visual area mapping, and custom validation with edge-case handling and BIDS compliance.
  • Automated QA: Automated daily biobank workflows, including a nightly per-subject validation dashboard and daily backup of newly processed and validated XNAT data to Box.
  • Cross-site QC: Compared MRI quality across Siemens CIMA.X, Prisma, NYUAD, and UAEU scanners using SNR, tSNR, CNR, and GSR metrics.
  • Normative dashboard: Preprocessed more than 9,000 HCP and ADNI subjects and built age- and gender-controlled normative curves for brain structure metrics.
  • Documentation: Authored technical documentation across 350+ commits and triaged and resolved user-reported GitHub issues.

LesionView & Longitudinal Report

Clinical MS lesion tracking and automated reporting

  • Built a clinical imaging application that tracks MS lesions across timepoints and reports new and enlarging lesions with color-coded and animated visualization.
  • Migrated a MATLAB prototype to a browser-based React Native/Expo application and iterated features weekly with Yas Clinic neurologists.
  • Co-inventor on provisional patent application NYU Ref. ABD01-03, filed May 2026.

CvsView

AI-ready central vein sign review for MS diagnosis

  • Built a browser-based review tool that helps clinicians efficiently verify the central vein sign in MS lesions, with a modular AI component designed for future models.
  • Resolved critical SWI-FLAIR alignment problems, containerized the application with Docker, and deployed it as an on-demand service for Harley Street Medical Centre.

MyPhysio

Full-stack physiotherapy practice management

  • Conceived and led development of a React Native, Expo, Next.js, and Supabase clinical application with admin dashboards, patient portals, exercise prescription, and pain and range-of-motion tracking.
  • Deployed with Hostinger, Supabase, GitHub Actions, and Docker.

afqView

Interactive 3D white matter tractography visualization

  • Built an interactive application for side-by-side tractography QC across 100+ subjects, 20+ tracts, and multiple processing iterations, with per-subject navigation and color-coded availability indicators.
